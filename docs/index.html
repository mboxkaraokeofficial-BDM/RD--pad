Roadmap

✅
เสร็จแล้ว
Phase 1 — Full UI + Mock Data
UI ครบทุกหน้า Now Playing, Search, Queue, Effects, Settings ด้วย Mock Data รองรับ Responsive ทุกขนาดหน้าจอ
🔄
กำลังพัฒนา
Phase 2 — Backend + TCP Bridge
Node.js WebSocket-to-TCP Bridge เชื่อมต่อ MBOX จริงผ่าน LAN ทดสอบคำสั่งควบคุมจริง (Play, Pause, Next, Volume, Key)
📋
แผนงาน
Phase 3 — Auth + Cloud DB (Supabase + LINE/Google)
ระบบ Login ด้วย LINE / Google OAuth, Supabase Database Cloud Song DB, Favorite Playlists, Admin Panel สำหรับร้านค้า
🔮
อนาคต
Phase 4 — DSP Voice Processing
Real-time DSP Voice Effects, AI Pitch Correction ประมวลผลเสียงขั้นสูงสำหรับนักร้องมืออาชีพ
